# 리액트로 과일주스를 팔아보자

### How to setting
```npm
git clone https://github.com/zkzofn/fruit-jucie.git
cd fruit-jucie
npm install
```

### How to start
```npm
npm start
```

### How to access to page
```npm
http://localhost:8080
or
http://localhost:8080/webpack-dev-server/
```
